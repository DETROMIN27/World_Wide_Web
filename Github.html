GitHub - найбільший веб-сервіс для хостингу IT-проектів та їх сумісний раз роботи.Веб-сервіс заснована на системі контролю версій Git та розроблений на Ruby on Rails та Erlang компанією GitHub, Inc (Раніше Logical Awesome). Сервіс безп латів для проектів з відкритим вихідним кодом та (з 2019 року) невеликих приватних проектів, надаючи їм все можливо сти (включаючи SSL), а для великих кор поративних проектів пропонуються раз особисті платні тарифні плани. ган сервісу — «Social Coding» ський можна перекласти як «Пишемо код разом». На футболках же друкують сов сім іншу фразу: "Fork you!" («Гілкувати тебе!»). З одного боку, вона співзвучна з англомовною лайкою натякає на неформальну атмосферу.З іншого, ці слова нагадують, що створювати нові форки з Git можна легко та безболісно — традиційно, до створення гілок разра ботчики проектів з відкритим вихідним кодом ставляться негативно - а також співзвучна назві однієї з можливо стей GitHub - черги форків
